Ejercicio #2 del examen de Front-End

El planteamiento es el siguiente:

• Crear una pagina que contenga un input donde el usuario pueda cargar tareas. También deberá tener un botón "guardar tarea" cuyo funcionamiento sea agregar el contenido del input a una lista sin eliminar los valores cargados previamente

• Crear un botón que se llame "Limpiar lista" y que al hacer click elimine todo el contenido de la lista de tareas.

• Agregar como mínimo dos estilos para los elementos que prefiera.
